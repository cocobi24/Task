# Trello project
Trello의 일부 기능을 구현한 프로젝트입니다.  
Back-end 에서는 Node.js로 서버 작업을 하였으며  
Front-end 작업은 Vanilla JS만으로 작업하였습니다.  

### 개발도구
Node.js(v14.16.1)  
Express(v4.17.1)  
MySQL 8.0  
ECMAScript 6  

### 구현 기능  
* 회원 기능 (가입,로그인/아웃,세션)
* 리스트 CRUD
* 카드 CRUD
* AWS RDS 데이터베이스 구축
* 로그인할 경우 서버에 리스트 및 카드 저장
