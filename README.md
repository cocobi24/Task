# Trello project
Trello의 일부 기능을 구현한 프로젝트입니다.  
Back-end 에서는 Node.js로 서버 작업을 하였으며  
Front-end 작업은 Vanilla JS만으로 작업하였습니다.  

### 개발도구
Node.js(v14.16.1)  
Express(v4.17.1)  
MySQL 8.0  
ECMAScript 6  

### 기능  
* 회원 기능 (가입,로그인/아웃,세션)
* 리스트 CRUD
* 카드 CRUD
	
### 기능 구현

- [x] user - 로그인 / 화원가입 
- [x] user - 회원 비밀 번호 변경 
- [x] user - 비밀번호 찾기 
- [x] list - 리스트 조회 
- [x] list - 리스트 추가 
- [x] list - 리스트 삭제 
- [x] list - 리스트 타이틀 수정 
- [x] card - 카드 조회
- [x] card - 카드 추가 
- [x] card - 카드 삭제 
- [x] card - 카드 수정 
- [x] card - 카드 이동
